升級 Buff 管理（局內三選一強化）

每個 buff＝一筆資料。效果由「對象＋鍵＋數值」決定：
・三圍(stat)：atk 攻 / def 防 / qir 氣勢率
・能力(mod)：maxHpAdd 最大HP / extraDraw 起手抽 / counterQi 相剋充氣勢 / regen 每回合回血 / ultDmg 輾壓傷害
・牌組(deck)：add 撿盤 / cullWeakest 丟最弱盤
（以上是程式已支援的效果鍵；要全新機制才需改程式碼。）
